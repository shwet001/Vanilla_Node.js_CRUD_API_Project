# Vanilla Node.js RestFul CRUD API Project
>This is vanilla Node.js CRUD API movie project.
## Routes
GET      /api/movies

POST     /api/movies

GET      /api/movies/:id

PUT      /api/movies/:id

DELETE   /api/movies/:id
# Installation
1. Install dependencies
   - npm install

2. Run in develpment
   - npm run dev

3. Run in production
   - npm start
